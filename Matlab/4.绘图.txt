基本绘图函数
	创建绘图
		plot 函数具有不同的形式，具体取决于输入参数
			如果 y 是向量，plot(y) 会生成 y 元素与 y 元素索引的分段线图。
			如果有两个向量被指定为参数，plot(x,y) 会生成 y 对 x 的图形。
		xlabel 添加x轴标签
		ylabel	添加y轴标签
		title	添加标题
	在一幅图形中绘制多个数据集
		通过调用一次 plot，多个 x-y 对组参数会创建多幅图形。matlab对每条线使用不同的颜色。
		legend 	简单标识各条线
	指定线型和颜色	
		plot(x,y,'color_style_marker')
			color_style_marker 包含一至四个字符（包括在单引号中），这些字符根据颜色、线型和标记类型构造而成
			如plot(x,y,'r:+')使用红色点线绘制数据，并在每个数据点处放置一个 + 标记
			
			颜色		'c'		青蓝
						'm'		品红
						'y'		黄
						'r'		红
						'g'		绿
						'b'		蓝
						'w'		白
						'k'		黑
			线型		'-'		实线
						'--'	虚线
						':'		点线
						'-.'	点划线
						无字符	没有线条
			标记类型	'+'		加号
						'o'		空心圆
						'*'		星号
						'x'		字母 x
						's'		空心正方形
						'd'		空心菱形
						'^'		空心上三角
						'v'		空心下三角
						'>'		空心右三角
						'<'		空心左三角
						'p'		空心五角形
						'h'		空心六角形
						无字符	无标记
	控制线条和标记
		如果指定标记类型，但未指定线型，则matlab仅使用标记创建图形，而不会创建线条
	绘制虚数和复数数据
		将多个复数值作为参数传递给 plot 时，MATLAB 会忽略虚部，但传递一个复数参数时除外
		axis equal 命令使 x 和 y 轴上的各刻度线增量的长度相同，这会使此绘图看起来更加圆润。
	将绘图添加到现有图形中
		hold 命令用于将绘图添加到现有图形中
		执行hold on，matlab不会再发出其他绘图命令时替换现有图形，而将新图形与当前图形结合在一起。
	图窗窗口
		如果尚未创建图窗窗口，绘图函数会自动打开一个新的图窗窗口。
		如果打开了多个图窗窗口，MATLAB 将使用指定为“当前图窗”（通常为上次使用的图窗）的图窗窗口
		要将现有图窗窗口设置为当前的图窗，请将指针放置在该窗口中并点击鼠标，或者也可以键入figure(n)。n 是图窗标题栏中的编号
		figure：打开新的图窗窗口并将其作为当前图窗
		
		清空图窗以便创建新绘图
			clf reset：创建新绘图，将此图窗的属性恢复为其默认值。
	在一幅图窗中显示多个绘图
		subplot 命令用于在同一窗口中显示多个绘图，或者在同一张纸上打印这些绘图
		subplot(m,n,p)：将图窗窗口划分为由多个小子图组成的 m×n 矩阵，并选择第 p 个子图作为当前绘图
	控制轴
		axis 命令提供了许多用于设置图形的比例、方向和纵横比的选项。
		自动改变坐标轴范围和刻度线
			默认情况下，MATLAB 查找数据的最大值和最小值，并选择坐标轴范围来覆盖此范围
			MATLAB 选择范围和轴刻度线值，以便生成可清楚地显示数据的图形
			可以使用 axis 或 xlim、ylim 与 zlim 函数来设置自己的范围
			axis manual：禁用极限自动设置。
			axis auto：启用自动极限选择
		设置坐标轴范围
			axis 命令用于指定自己的极限
				如axis([xmin xmax ymin ymax]) 或者axis([xmin xmax ymin ymax zmin zmax])
		设置轴纵横比
			axis square：使x轴和y轴长度相同
			axis equal：使x轴和y轴上的各个刻度线增量的长度相同
		设置轴可见性
			axis on：显示轴
			axis off：隐藏轴
		设置网格线
			grid on：启用网格线
			grid off：禁用网格线
	添加轴标签和标题
		..
		text:在图形中添加文本附注
	保存图窗
		默认保存为.fig文件
		savefig：将图窗及包含的图形对象保存为.fig文件 
		saves：以各种格式保存图窗
		
		加载图窗
			openfig：加载保存为.fig文件的图窗
			imread：将标准图形文件(包括图窗)加载到matlab中
	保存工作区数据
		save 保存为.mat文件
		
		
创建网格图和曲面图
	关于网格图和曲面图
		在 x-y 平面中的网格上方使用点的 z 坐标来定义曲面图，并使用直线连接相邻的点。
		mesh 函数生成仅使用颜色来标记连接定义点的线条的线框曲面图
		surf 函数使用颜色显示曲面图的连接线和面
		matlab通过将索引的 z 数据值映射到图窗颜色图来标记曲面图颜色
	可视化包含两个变量的函数
		要显示包含两个变量的函数 z = f (x,y)，
			在此函数的域中，生成分别由重复行和重复列组成的 X 和 Y 矩阵。
			使用 X 和 Y 计算此函数并绘制其图形。
		meshgrid 函数将一个向量或两个向量（即 x 和 y）指定的域转换为矩阵 X 和 Y，以便用于计算包含两个变量的函数。
		X 的行是向量 x 的副本，Y 的列是向量 y 的副本。

		绘制正弦函数
		彩色曲面图
			曲面图与网格图相似，但曲面图的矩形面是彩色的。各个面的颜色由 Z 的值及颜色图（即颜色的有序列表）决定
		将曲面图设置为透明
		使用灯光照亮曲面图

		
显示图像
	图像数据
		可以将二维数值数值显示为图像。
		在图像中，数组元素决定了图像的亮度或颜色。
		imshow：显示图像
	读取和写入图像
		imread 
		imwrite

		
打印图形
	可以直接打印图窗，也可以打印导出后的某种标准图形文件格式。
	print：通过命令行来打印或导出图窗
	

处理图形对象
	图形对象
		图形对象是用于显示图形的基本元素。
		常见图形对象
			图窗 - 包含轴、工具栏、菜单等的窗口。
			轴 - 包含表示数据的对象的坐标系
			线条 - 代表传递至 plot 函数的数据值的线条。
			文本 - 用于轴刻度线和可选标题及注释的标签。
		访问对象属性
			.属性
		查找对象的属性
			get(对象)
	设置对象属性
		set 
		设置现有对象的属性
		设置多个属性值
	用于处理对象的函数
		allchild		查找指定对象的所有子级
		ancestor		查找图形对象的父级
		copyobj			复制图形对象
		delete			删除对象
		findall			查找所有图形对象（包括隐藏句柄）
		findobj			查找具有指定属性值的对象的句柄
		gca				返回当前轴的句柄
		gcf				返回当前图窗的句柄
		gco				返回当前对象的句柄
		get				查询对象的属性的值
		ishandle		如果值是有效对象句柄，则为 True
		set				设置对象的属性的值
	传递参数 
	查找先有对象的句柄
		findobj
		查找特定类型的所有对象
		查找具有特定属性的对象
