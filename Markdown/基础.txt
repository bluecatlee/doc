标题
	1.使用 = 和 - 标记一级和二级标题
		=表示一级标题 即h1
		-表示二级标题 即h2
		必须在标题换行后使用，至少两个标记才会生效
		如果反复出现=和-，则以最后一个标记为准
		
		注意如果-标记的上一行是是空行或者没有行，则三个及以上-标记会展现成水平线
		
	2.使用 # 号标记
		使用 # 号可表示 1-6 级标题，一级标题对应一个 # 号，二级标题对应两个 # 号，以此类推
		超过6个#将以普通文本展示
		注意#号后于标题内容之间有一个空格 否则将无效
		
		
段落格式
	1.段落 
		段落没有特殊的格式，直接编写文字。
		段落的换行方式
			使用两个(及以上)空格加上回车 (区块的情况下两个空格就换行了)
			一个tab加上回车也能换行
			空行(即两个回车，顾名思义这种方式就会多个空行)
		注意单个换行符、或者n个tab、或者n个空格的效果都是一个空格
		
	2.字体 
		*斜体文本*
		_斜体文本_
		**粗体文本**
		__粗体文本__
		***粗斜体文本***
		___粗斜体文本___
		
	3.分隔线
		可以在一行中用三个以上的星号、减号、底线来建立一个分隔线，行内不能有其他东西(除了空格)
	4.删除线	
		~~BAIDU.COM~~
	5.下划线
		通过 HTML 的 <u> 标签来实现
		<u>带下划线文本</u>
	6.脚注
		[^要注明的文本]
		

列表 
	无序列表使用星号(*)、加号(+)或是减号(-)作为列表标记，这些标记后面要添加一个空格，然后再填写内容
		* 第一项
		* 第二项
		* 第三项
	有序列表使用数字并加上 . 号来表示 (其实数字的大小并不影响顺序 只是个格式)
		1. 第一项
		2. 第二项
		3. 第三项
	支持列表嵌套
	

区块
	区块引用是在段落开头使用 > 符号 ，然后后面紧跟一个空格符号
	区块是可以嵌套的，一个 > 符号是最外层，两个 > 符号是第一层嵌套，以此类推
	区块中支持使用列表	
	
	
代码
	如果是段落上的一个函数或片段的代码可以用反引号把它包起来（`）
	代码区块
		使用 4 个空格或者一个制表符（Tab 键）
		也可以用 ``` 包裹一段代码，并指定一种语言（也可以不指定）
		
		
链接
	[链接名称](链接地址)  或者  <链接地址>	
	高级链接
		可以通过变量来设置一个链接，变量赋值在文档末尾进行
		
		
图片 
	![alt 属性文本](图片地址)

	![alt 属性文本](图片地址 "可选标题")
	
	也可以像网址那样对图片网址使用变量
	
	不能指定图片的高度与宽度，如果需要的话可以使用普通的 <img> 标签
	
	
表格 
	制作表格使用 | 来分隔不同的单元格，使用 - 来分隔表头和其他行
		|  表头   | 表头  |
		|  ----   | ----  |
		| 单元格  | 单元格 |
		| 单元格  | 单元格 |
		
		-: 设置内容和标题栏居右对齐。
		:- 设置内容和标题栏居左对齐。
		:-: 设置内容和标题栏居中对齐。
		
		
高级
	支持的HTML元素
		不在 Markdown 涵盖范围之内的标签，都可以直接在文档里面用 HTML 撰写。
		目前支持的 HTML 元素有：<kbd> <b> <i> <em> <sup> <sub> <br>等
	
	转义
		使用反斜杠转义特殊字符
		
	公式 
		使用两个美元符 $$ 包裹 TeX 或 LaTeX 格式的数学公式来实现
		
	代码 
		可以画流程图、时序图等
